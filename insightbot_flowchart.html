🔄 Auto-Recovery Loop (on Databricks error)
Classify error type → Rewrite SQL (LLM) → Retry (max 2×)
Error types: column not found · table not found · syntax error · timeout · no rows
If exhausted: Suggestion Engine finds nearest answerable alternative
📊 Anomaly Thresholds
  • Delivery > 20 days
  • Cancellation rate > 5%
  • Revenue drop > 10% MoM
  • Seller review < 3.0 / 5
Up to 3 unique flags surfaced per response
💾 User replies download → full CSV uploaded to Slack
